Working on exercises

Useful commands

  • npm start - start all 3 built-in servers: slides, exercises & db
  • npm test - run unit tests in tdd mode
  • npm run exercises - start angular exercises on port 4200
  • npm run slides - start only slides on port 8080
  • npm run db - start only database server on port 3000

Working on exercises

Visual Studio Code


  • Install workspace recommended extensions
  • Change keymap in 'Preferences / Keymap Extensions'
  • Enable VSCode icons in 'Preferences / File Icon Theme'
  • Use workspace TypeScript version, not the VS Code's one
  • Careful with auto-imports → check ex. number you're importing from
